Chain Abstractions


This poem is an example output from a program I built. My plan is to have these poem lines be populated from a Flask API call, then to display them line by line like so. I also plan to add distored images that relate to the items mentioned in the poem.

to milk

    is to take advantage of,

          but to milkshake is

a shake

    is an ice cream treat,

          but a shakedown is

to be down

    is to be not up,

          but to be downbeat is

to be beat?
